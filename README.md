# Solana Skills Bundle for Kimi CLI

A comprehensive collection of Solana blockchain skills for Kimi CLI, covering development, operations, and DeFi integrations.

## 📦 What's Included

### 1. solana-development.skill
Complete Solana program development guide with:
- Anchor Framework 0.32+ patterns
- Native Rust program development
- SPL Token & Token-2022 operations
- LiteSVM testing framework
- Security best practices
- @solana/kit v2 client examples

### 2. solana-operations.skill 
Master Solana infrastructure with:
- Solana CLI complete reference
- Helius API integration (enhanced RPC, webhooks)
- Chainstack enterprise RPC setup
- Wallet management & staking operations
- Transaction monitoring

### 3. solana-defi.skill 
Build DeFi applications with:
- Jupiter V6 API (swaps, limit orders, DCA)
- Raydium AMM & CLMM patterns
- Meteora DLMM integration
- Jito MEV protection bundles
- Priority fee optimization

## 🚀 Installation

```bash
# Install each skill
cp solana-development.skill ~/.kimi/skills/
cp solana-operations.skill ~/.kimi/skills/
cp solana-defi.skill ~/.kimi/skills/

# Extract
cd ~/.kimi/skills && \
  tar -xzf solana-development.skill && \
  tar -xzf solana-operations.skill && \
  tar -xzf solana-defi.skill
```

## 💡 Usage Examples

- **"Create an Anchor program for a token vault"** → solana-development
- **"Check SOL balance and send transaction"** → solana-operations
- **"Swap tokens using Jupiter"** → solana-defi
